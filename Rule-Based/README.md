# Reactive Agent

## How to Run
1. Install the required packages:
```
pymupdf python-docx
```
2. Place the following files in the same folder:
   - `main.py`
   - `jobs.json`
   - Candidate CV (`.pdf`, `.docx`, or `.txt`)

3. Run the program:
```bash
python main.py
```
4. Enter the CV file path when prompted.

This implementation is a pure rule-based Reactive Agent and does not use any LLM or external AI service.
